## 任务点

### 添加内链

- [[: 选择已经存在的笔记，如果输入不存在的笔记也会生成内链但不会创建，需要使用 alt + enter 创建
- [[#: 选择笔记中的某个标题
- [[|: 通过 | 修改在预览中显示名称

### 预览双链

需要打开 `Core plugins - Page preview`

- cmd + hover

### 添加文件

- ![[: 添加当前空间的文件
- 使用命令：Insert attachment：在磁盘选择文件，会把该文件放到当前空间

### 在新窗格打开链接

- cmd + opt + enter
- cmd + click

### 在当前窗格打开链接

- opt + enter

### note refactor 插件

- cmd + shift + c: 添加到笔记里面
- cmd + shift + n: 新建笔记
