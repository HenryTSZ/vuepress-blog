## 任务点

### 打断点

- cmd + b: 打开/关闭断点
- cmd + shift + b: 切换断点是否可用

### step over

- cmd + '
- F10

### step into

- cmd + ;
- F11

### 停止当前断点

- F5
- cmd + \

### 切换全部断点是否可用

- cmd + F8

### 输出选中的代码

- cmd + shift + e

### 跳转到函数

- cmd + shift + o

### 跳转到指定行

- ctrl + g
