### 选择文字

VC:

- yv: 进入可视模式，并全选指定文本
- V: 选择文字，对齐到行
- y: 复制选择的文字

[改键](https://github.com/gdh1995/vimium-c/wiki/Use-in-another-keyboard-layout)：

```
mapkey <L:v> <$>
mapkey <H:v> <0>
```

SK:

- zv: 进入可视模式，并全选指定文本
- v: 进入可视模式，通过屏幕中的字母选择位置后，需要再按一次 v 才能选择
- y: 复制选择的文字

改键：

```
api.vmap('L', "$");
api.vmap('H', "0");
```
