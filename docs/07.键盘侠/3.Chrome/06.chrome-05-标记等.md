## 任务点

### 创建一个标记

VC:

- m: 在当前页创建一个滚动位置标记
- `: 跳转到指定标记

只在当前页面生效。比如在当前页面当前位置使用 `mm` 创建一个标记，滚动到别的位置，可以使用 \`m 使页面滚动到标记位置，切换到别的页面就不生效了

SK:

- m: 为当前 URL 设置类 vim 标示
- ': 访问类 vim 标签

跨页面生效，只能跳转到页面，不能滚动到标记位置。比如在当前页面使用 `mm` 创建一个标记，切换到别的页面，使用 `'m` 可以跳转回来；而当前页面滚动位置变化后，`'m` 会打开新页面
