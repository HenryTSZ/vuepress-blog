## 任务点

### 移动光标到行首

- cmd + left
- ctrl + a

### 移动光标到行尾

- cmd + right
- ctrl + e

### 向左移动一个单词

- opt + left

### 向右移动一个单词

- opt + right

### 删除一个单词

- opt + delete

### 基于光标向右删除

- ctrl + d

### 选中到行首

- shift + up
- shift + cmd + left

### 选中到行尾

- shift + down
- shift + cmd + right

### 向左选中

- shift + left

### 向右选中

- shift + right

### 向左选中一个单词

- shift + opt + left

### 向右选中一个单词

- shift + opt + right

### 移动到上一行

- ctrl + p

### 移动到下一行

- ctrl + n

### 光标左移动

- ctrl + b

### 光标右移动

- ctrl + f
