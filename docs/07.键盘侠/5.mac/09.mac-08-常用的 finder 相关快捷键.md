## 任务点

### 选择

- 方向键

### 打开，类似双击

- cmd + 下方向键

### 返回上一级

- cmd + 上方向键

### 文件夹历史记录前进

- cmd + ]

### 文件夹历史记录后退

- cmd + [

### 复制

- cmd + d

### 查看信息

- cmd + i

### 新建文件夹

- cmd + shift + n

### 删除

- cmd + delete

### 打开一个新的 Finder

- cmd + n

### 拖拽后复制

- opt
