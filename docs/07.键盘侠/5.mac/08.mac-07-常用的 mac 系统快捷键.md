## 任务点

### 打开 app 的设置

- cmd + ,

### 退出当前的应用

- cmd + q

### 锁屏

- cmd + ctrl + q

### 隐藏当前窗口

- cmd + h

### 最小化当前窗口

- cmd + m

### 同一个 APP 多窗口之间切换

- cmd + `；改键：cmd + esc

### 切换到最近使用的 app

- cmd + tab
- cmd + shift + tab

### 全屏

- cmd + ctrl + f

### 切换桌面

- ctrl + 左右方向键
