### 使用 snippets

### 重构提示快捷键

- ctrl + shift + r: 显示和重构相关的命令
- cmd + .: 包含重构及其他命令

### 常用重构命令

- 重命名
- 提炼一个函数（全局、内部）
- 提炼一个变量，即常量变变量
- 反提炼一个变量，即变量变常量
- 引号变为模版字符串
- 创建函数
- 创建变量
- function 函数与箭头函数互转
- if/else 与三元表达式互转

内外平衡：

JavaScript Booster: 使用 `javascriptBooster.extendSelection and javascriptBooster.shrinkSelection`

vim: 使用 `vaf` 或 `vii`
