映射命令：

```json
"vim.visualModeKeyBindings": [
  {
    "before": ["<Leader>", "/"],
    "after": ["y", "/", "<C-r>", "\"", "<Enter>"]
  }
],
"vim.normalModeKeyBindings": [
  {
    "before": ["<Leader>", "/"],
    "after": ["/", "<C-r>", "\"", "<Enter>"]
  }
]
```

在 `visual` 模式，在选中文本后，按下 `<Leader> /` 就可以快速搜索了；在 `normal` 模式，按下 `<Leader> /` 就可以快速搜索刚才复制或删除的文本
