## 任务点

### 左右分屏

- cmd + d

### 上下分屏

- cmd + shift + d

### 切换分屏

- cmd + opt + 方向键
- cmd + [/]: 轮流切换

### 调整窗口

- cmd + ctrl + 方向键

### 新建标签页

- cmd + t

### 关闭标签页

- cmd + w: 会把标签页的分屏一并关闭

### 切换标签页

- cmd + 1/2/3...: 切换到第几个标签页
- cmd + 方向键

### 移动标签页

- cmd + shift + 方向键
