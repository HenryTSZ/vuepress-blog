在上一个技巧中，我们想为每个 `+` 号的前后各添加一个空格。我们先用 `f+` 命令跳到目标字符上，然后用 `s` 命令把一个字符替换成 3 个，做完这步准备后，我们就可以按若干次 `;.` 完成此任务。

理想模式：用一键移动，另一键执行

用一次按键移动，另一次按键执行，没有再比这更好的了，不是吗？这就是我们的理想解决方案。我们将会一次又一次地看到这一编辑模式，所以为了方便起见，我们把它叫做“. 范式”。
