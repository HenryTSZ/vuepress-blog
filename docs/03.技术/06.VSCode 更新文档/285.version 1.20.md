### ["Smart Case" search](https://code.visualstudio.com/updates/v1_20#_smart-case-search)

You can now setÂ `"search.smartCase": true`Â to enable "smart case" mode for global search. When enabled, VS Code will automatically do a case-sensitive search when you search for a query that contains an uppercase character. If your search query is all lowercase, then the search will be case-insensitive.

For example, searching "code" will match "code" or "Code". Searching "Code" will only match "Code".

### [New keyboard shortcut context operator](https://code.visualstudio.com/updates/v1_20#_new-keyboard-shortcut-context-operator)

In this release, there is a new key-value pair operator forÂ `when`Â clauses. The expressionÂ `key =~ value`Â treats the right hand side as a regular expression to match against the left hand side. For example, to contribute context menu items for all Docker files, one could use:

```

```
