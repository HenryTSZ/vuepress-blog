## React 简介

**`React` 是什么?**

`React` 是用于构建用户界面的 `JS` 库. 是一个将数据渲染为 `HTML` 视图的开源 `JS` 库.

**为什么学?**

1. 原生 JS 操作 DOM 繁琐, 效率低

2. 使用 JS 直接操作 DOM, 浏览器会进行大量的重绘重排

3. 原生 JS 没有组件化编码方案, 代码复用低
