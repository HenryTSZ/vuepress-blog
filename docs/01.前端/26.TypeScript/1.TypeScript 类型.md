## 字面量

1. 有点类似 const

    ```ts
    let a: 10
    a = 11 // 报错: Type '11' is not assignable to type '10'.
    ```
2. 限制值的范围

    当一个变量的取值在某些范围内时, 可以使用字面量

    比如 true | false, male | female

    ```ts
    let b: 'male' | 'female'
    b = 'male'
    b = 'female'
    b = 'hello' // 报错: Type '"hello"' is not assignable to type '"male" | "female"'.
    ```
