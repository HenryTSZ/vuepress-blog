## 快捷键

注: `+` 表示需要点击下一个键时, 上一个或多个键仍是按下去状态; `空格` 表示点击下一个键时, 上一个或多个键可以放开

- C: cmd / ctrl
- S: shift
- O: option / alt
- D: Delete(Mac 自带键盘没有)
- B: Backspace(Mac 自带键盘的 delete)
- E: Enter(Mac 自带键盘的 return)
- H: Home
- Ed: End
- ↑: UpArrow(上箭头)
- ↓: DownArrow(下箭头)
- ←: LeftArrow(左箭头)
- →: RightArrow(右箭头)
- CT: control / ctrl
- T: tab
